# Anti-scam-bot

## Requirements:
1. Nodejs>=16
2. Git
3. A Discord bot token
## Installing:
1. Run `git clone https://github.com/HELLSNAKES/anti-scam-bot.git`
2. Run `cd anti-scam-bot`
3. Run `npm i`
4. Rename `.env_example` to `.env` 
5. Run `npm start` to get the bot working
